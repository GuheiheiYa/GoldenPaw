## 三、新增页面

| 页面 | 路由 | 功能 |
|------|------|------|
| 设置页面 | `/pages/settings/index?type=xxx` | 通用设置页面，支持11种类型 |
| 搜索页面 | `/pages/search/search` | 按备注、分类、金额搜索交易 |
| 帮助页面 | `/pages/help/help` | FAQ + 联系方式 |

### 设置页面支持的类型
- category - 分类管理
- account - 账户管理
- theme - 主题颜色
- reminder - 提醒设置
- security - 密码/指纹锁
- currency - 币种设置
- cycle - 记账周期
- export - 导出数据
- import - 导入数据
- sync - 云同步
- clear - 清空数据
