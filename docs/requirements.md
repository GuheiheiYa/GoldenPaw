# GoldenPaw 需求文档

## 功能模块总览

| 模块 | 功能 | 状态 |
|------|------|------|
| 首页 | 余额展示、快捷操作、周历、预算、存钱目标、最近流水 | ✅ 已完成 |
| AI记账 | 语音记账、聊天记账、票据识别、智能分析 | ⚠️ 核心功能完成 |
| 资产 | 净资产、资产构成、账户列表、资产趋势 | ⚠️ 部分完成 |
| 我的 | 用户信息、数据管理、个性化设置 | ⚠️ 部分完成 |
| 记账（+号） | 金额输入、分类选择、收支切换、备注信息 | ✅ 已完成 |
| 收支报表 | 趋势图表、饼图、消费排行 | ✅ 已完成 |
| 明细 | 交易列表、类型筛选、日期范围筛选、搜索 | ✅ 已完成 |
| 存钱目标 | 目标列表、预算管理 | ✅ 已完成 |
| 待确认 | 自动监听记录确认、来源设置 | ⚠️ 视觉完成，逻辑待补 |

## 详细需求

### 1. 首页

**功能点：**
- [x] 顶部品牌标识 + 搜索/消息按钮
- [x] 本月结余卡片（收入、支出、结余）
- [x] 快捷操作胶囊（报表、明细、存钱、待确认）
- [x] 周历组件（显示本周日期，高亮今天）
- [x] 本月预算卡片（分类预算进度条）
- [x] 存钱目标卡片（目标进度）
- [x] 最近流水列表（8条）
- [x] 自定义 TabBar + 悬浮记账按钮

### 2. AI记账

**功能点：**
- [x] AI助手状态卡片
- [x] 快捷操作网格（语音记账、聊天记账、票据识别、智能分析）
- [x] 聊天界面（消息气泡、建议芯片）
- [x] 输入框 + 发送按钮
- [x] 自然语言解析记账（金额/分类/收支类型识别）
- [x] 智能分析查询（本月收支、累计收支）
- [ ] 语音输入（需要语音识别 API）
- [ ] 票据识别（OCR，需要图像识别 API）

### 3. 资产

### 4. 我的

**功能点：**
- [x] 用户卡片（头像、昵称、ID、统计数据）
- [x] 数据管理菜单（导出、导入、云同步、清空）
- [x] 个性化菜单（主题、提醒、密码锁）
- [x] 基础设置菜单（分类、账户、币种、周期）
- [x] 关于菜单

**子页面状态：**
- [x] 分类管理：添加/删除自定义分类（底部弹窗表单，默认分类不可删）
- [x] 账户管理：添加账户（名称/类型/银行/图标）
- [ ] 主题颜色：UI 框架，未实现多套配色
- [x] 提醒设置：记账提醒 + 预算预警开关（持久化）
- [ ] 密码/指纹锁：toast 占位
- [ ] 币种设置：toast 占位
- [ ] 记账周期：toast 占位
- [x] 导出数据：CSV 导出（H5 下载，其他平台复制剪贴板）
- [ ] 导入数据：toast 占位
- [ ] 云同步：toast 占位
- [x] 清空数据：确认弹窗 + 真实清空
