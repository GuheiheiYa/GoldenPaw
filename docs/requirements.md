# GoldenPaw 需求文档

## 功能模块总览

| 模块 | 功能 | 状态 |
|------|------|------|
| 首页 | 余额展示、快捷操作、周历、预算、存钱目标、最近流水 | 未开始 |
| AI记账 | 语音记账、聊天记账、票据识别、智能分析 | 未开始 |
| 资产 | 净资产、资产构成、账户列表、资产趋势 | 未开始 |
| 我的 | 用户信息、数据管理、个性化设置 | 未开始 |
| 记账（+号） | 金额输入、分类选择、收支切换、备注信息 | 未开始 |
| 收支报表 | 趋势图表、饼图、消费排行 | 未开始 |
| 明细 | 交易列表、筛选、搜索 | 未开始 |
| 存钱目标 | 目标列表、预算管理 | 未开始 |
| 待确认 | 自动监听记录确认、来源设置 | 未开始 |

## 详细需求

### 1. 首页

**功能点：**
- [ ] 顶部品牌标识 + 搜索/消息按钮
- [ ] 本月结余卡片（收入、支出、结余）
- [ ] 快捷操作胶囊（报表、明细、存钱、待确认）
- [ ] 周历组件（显示本周日期，高亮今天）
- [ ] 本月预算卡片（分类预算进度条）
- [ ] 存钱目标卡片（目标进度）
- [ ] 最近流水列表（4条）
- [ ] 自定义 TabBar + 悬浮记账按钮

### 2. AI记账

**功能点：**
- [ ] AI助手状态卡片
- [ ] 快捷操作网格（语音记账、聊天记账、票据识别、智能分析）
- [ ] 聊天界面（消息气泡、建议芯片）
- [ ] 输入框 + 发送按钮

### 3. 资产

**功能点：**
- [ ] 净资产卡片（金额、变化趋势）
- [ ] 资产构成环形图
- [ ] 账户列表（储蓄卡、基金、股票、现金、信用卡）
- [ ] 添加账户按钮
- [ ] 资产趋势柱状图

### 4. 我的

**功能点：**
- [ ] 用户卡片（头像、昵称、ID、统计数据）
- [ ] 数据管理菜单（导出、导入、云同步、清空）
- [ ] 个性化菜单（主题、提醒、密码锁）
- [ ] 基础设置菜单（分类、账户、币种、周期）
- [ ] 关于菜单

### 5. 记账 BottomSheet（+号按钮）

**功能点：**
- [ ] 底部弹出面板（动画）
- [ ] 支出/收入/转账 Segmented 切换
- [ ] 金额输入（自定义数字键盘）
- [ ] 分类选择网格（2行，可滚动）
- [ ] 日期选择（默认今天）
- [ ] 账户选择（默认常用）
- [ ] 标签选择
- [ ] 备注输入
- [ ] 完成按钮 → 保存交易

### 6. 收支报表

**功能点：**
- [ ] 时间周期选择（本周/本月/本季/本年/自定义）
- [ ] 收支摘要卡片
- [ ] 收支趋势柱状图
- [ ] 支出/收入饼图
- [ ] 消费排行 TOP 5

### 7. 明细

**功能点：**
- [ ] 筛选Tab（全部/支出/收入/转账）
- [ ] 搜索按钮
- [ ] 筛选按钮
- [ ] 月度分组交易列表
- [ ] 交易项（图标、标题、分类、金额、标签）
- [ ] 左滑编辑/删除

### 8. 存钱目标

**功能点：**
- [ ] Segmented 切换（存钱目标/预算管理）
- [ ] 目标卡片（图标、名称、进度环、进度条、统计数据）
- [ ] 存入金额 / 调整目标 操作按钮
- [ ] 预算管理（总预算、分类预算列表）
- [ ] 新建目标/预算

### 9. 待确认

**功能点：**
- [ ] 信息提示横幅
- [ ] 待确认卡片（来源、原始文本、解析结果）
- [ ] 确认/编辑/忽略 操作按钮
- [ ] 监听来源设置（开关切换）
